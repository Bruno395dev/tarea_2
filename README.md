## Cómo ejecutar el proyecto

### 1. Clonar el repositorio

```bash
git clone https://github.com/Bruno395dev/tarea_2.git
```

### 2. Dependencias

Este proyecto no requiere instalar dependencias.

### 3. Ejecutar en el navegador con Live Server

1. Abrir la carpeta del proyecto en **Visual Studio Code**.
2. Asegurarse de tener instalada la extensión **Live Server** (`ritwickdey.LiveServer`).
3. Hacer clic derecho sobre `index.html` y seleccionar **"Open with Live Server"**, o presionar el botón **"Go Live"** en la barra de estado inferior de VS Code.
4. El proyecto se abrirá automáticamente en el navegador en `http://127.0.0.1:5500`.
